# Criando-Mapa-de-Habilidades-
Como desafio da Dio vamos identificar minhas habilidades através do Mapa de Habilidades, fazer um cruzamento com as necessidades do mercado e indicar caminhos para monetização dessas habilidades
Primeriro identificamos minhas habilidades com base nessas 5 perguntas com auxilio do Copilot:

1 O que as pessoas costumam elogiar em você?
2 Em que tipo de tarefa você se sente confiante?
3 Que tipo de ajuda você costuma oferecer para amigos ou familiares?
4 Quais atividades você faz com facilidade, enquanto outras pessoas têm dificuldade?
5 Que tipo de conteúdo você consome com frequência? Isso pode indicar interesse e habilidade.

A respostas geraram o quadro de habilidades/competencias abaixo:
